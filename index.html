Simple by design

Four steps. One evening.
Whole class done.

Each step takes seconds. The whole class takes one evening instead of three nights.

📋
01

Add your class list

Enter your learners' names once. MyReportComments remembers them for the whole session and across devices with the web app.

☑️
02

Pick from dropdowns

For each learner, select from simple pick lists — performance, potential, behaviour, peers, sport. Type in subjects they excelled in or need to focus on.

⚡
03

Hit Generate

A polished, professional, positively-worded comment appears in under three seconds. Not happy with one sentence? Click it to reshuffle just that part. Or edit the text directly.

📤
04

Export the whole class

When you're done, export every comment in one click. Ready to paste into your school's system or print directly.

What you get

Everything report season
actually demands.

Built from real report-writing frustrations. Every feature exists because a teacher needed it.

✦

Always Unique

9 criteria × 3 options per criterion × free text subjects = thousands of distinct comment combinations. No two learners in your class will ever receive the same wording.

🔄

3-Month Memory

MyReportComments tracks which comments have been used and avoids repeating them across terms — essential for SA schools running four report cycles a year.

💬

Positively Phrased

Every comment — even for learners who need more support — is framed constructively and warmly. No deficit language, ever. Consistent with CAPS reporting conventions.

🔀

Sentence Reshuffling

Not happy with one sentence? Click it to reshuffle just that part. Every line can be regenerated independently without losing the rest of the comment.

📐

Length Control

Choose exactly how long each comment should be — short, medium, or detailed. The tool respects your school's word count requirements automatically.

🧑‍🎓

Pronoun Aware

Set the correct pronoun for each learner and every generated comment uses it consistently throughout — he/him, she/her, or they/them.

📋

Class Manager

Your full class list lives inside the tool. Generate, review, and store a comment per learner — then come back to any of them before you export.

📤

One-Click Export

Export your completed class comments to a formatted document in one click. No copy-pasting between tabs, no manual formatting required.

✏️

Always Editable

Every generated comment is yours to refine. Edit the text directly before copying. The tool gives you a first draft that's 90% done — you finish the last 10%.
